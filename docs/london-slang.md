# London Slang - Expressões Usadas por Jovens em Londres

## Gírias Essenciais de Londres

| Slang | Significado | Exemplo de Uso |
|-------|-------------|----------------|
| Bare | Muito, bastante | "There's bare people here" |
| Sick | Legal, incrível | "That's sick, mate!" |
| Peng | Bonito/a, atraente | "She's well peng" |
| Fam | Família, amigos próximos | "What's up, fam?" |
| Init | Não é? (isn't it) | "That's mad, init?" |
| Safe | Legal, tranquilo | "Safe, bruv!" |
| Wicked | Incrível, fantástico | "That party was wicked!" |
| Bruv | Irmão, amigo | "Alright, bruv?" |
| Blud | Amigo, parceiro | "Listen, blud..." |
| Ends | Bairro, vizinhança | "I'm from South London ends" |
| Garms | Roupas | "Nice garms, mate!" |
| Chirpsing | Flertando | "He's chirpsing that girl" |
| Bait | Óbvio, evidente | "That's so bait!" |
| Bangin' | Muito bom | "This food is bangin'" |
| Bovvered | Incomodado | "I ain't bovvered" |
| Chung | Muito bonito/a | "He's proper chung" |
| Dry | Chato, sem graça | "This party is dry" |
| Hench | Musculoso | "He's well hench" |
| Jokes | Engraçado | "That was jokes!" |
| Kotch | Relaxar, descansar | "Let's kotch at mine" |
| Link | Encontrar alguém | "Let's link up later" |
| Long | Trabalhoso, cansativo | "That's too long, fam" |
| Marvin | Com fome (Marvin Gaye = starving) | "I'm proper Marvin" |
| Rinsed | Usado demais | "That song is rinsed" |
| Vexed | Irritado | "I'm so vexed right now" |
| Wag-One | O que está acontecendo? | "Wag-one, fam?" |
| Yard | Casa | "Come to my yard" |
| Allow it | Para com isso | "Allow it, bruv!" |
| Butters | Feio/a | "That's butters" |
| Dapper | Elegante | "Looking dapper today!" |
