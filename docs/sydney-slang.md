# Sydney/Australian Slang - Expressões Usadas por Jovens na Austrália

## Gírias Essenciais Australianas

| Slang | Significado | Exemplo de Uso |
|-------|-------------|----------------|
| G'day | Olá | "G'day, mate! How are you?" |
| Mate | Amigo, parceiro | "Thanks, mate!" |
| Arvo | Tarde (afternoon) | "See you this arvo!" |
| Barbie | Churrasco | "Let's have a barbie!" |
| Heaps | Muito, bastante | "That's heaps good!" |
| Reckon | Achar, pensar | "I reckon it'll rain today" |
| No worries | Sem problemas | "No worries, mate!" |
| She'll be right | Vai ficar tudo bem | "She'll be right, don't stress" |
| Fair dinkum | Verdade, sério | "Fair dinkum? That's crazy!" |
| Crikey | Expressão de surpresa | "Crikey! That was close!" |
| Stoked | Muito animado | "I'm stoked about the trip!" |
| Chuffed | Muito feliz | "I'm chuffed with the result" |
| Keen | Interessado, animado | "I'm keen to go!" |
| Suss | Suspeito | "That looks a bit suss" |
| Dodgy | Duvidoso | "That deal sounds dodgy" |
| Bogan | Pessoa sem sofisticação | "He's a bit of a bogan" |
| Sickie | Falta por doença (falsa) | "I'm chucking a sickie today" |
| Servo | Posto de gasolina | "I'll stop at the servo" |
| Bottle-o | Loja de bebidas | "Grab some drinks at the bottle-o" |
| Thongs | Chinelos | "Wear your thongs to the beach" |
| Esky | Caixa térmica | "Grab a beer from the esky" |
| Snag | Salsicha | "Want a snag on the barbie?" |
| Sanga | Sanduíche | "I made a sanga for lunch" |
| Lollies | Doces | "I bought some lollies" |
| Dunny | Banheiro | "Where's the dunny?" |
| Ute | Caminhonete | "I'll take the ute" |
| Tinny | Latinha de cerveja | "Pass me a tinny" |
| Legless | Muito bêbado | "He got legless last night" |
| Hooroo | Tchau | "Hooroo! See ya later!" |
| Yonks | Muito tempo | "It's been yonks since I saw you!" |

## Frases Típicas Australianas

| Frase | Significado | Contexto |
|-------|-------------|----------|
| "No worries, mate!" | Sem problemas! | Resposta a agradecimento |
| "She'll be right" | Vai ficar tudo bem | Tranquilizar alguém |
| "Too easy!" | Muito fácil! | Quando algo é simples |
| "Good on ya!" | Muito bem! | Parabenizar alguém |
| "You beauty!" | Que maravilha! | Celebrar algo bom |
| "Strewth!" | Nossa! | Expressão de surpresa |
| "Chuck a U-ey" | Fazer retorno | Direção de trânsito |
| "I'm flat out" | Estou muito ocupado | Descrever rotina |
