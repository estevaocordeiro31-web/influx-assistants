| Expressão | Explicação | Exemplo |

| measurements with "of" | (a depth / a length / a width / a height / etc.) of (3 feet / 2 meters / 10 inches / etc.) | - |
| boundary | to draw / to fix / to mark / to cross / to establish / etc. the boundary | - |
| color something | you color something red / yellow / green / etc. | The kid drew a car and colored it blue. |

| -ish (suffix) | = approximately = sort of = kind of | reddish / yellowish / eightish / youngish / etc. |
