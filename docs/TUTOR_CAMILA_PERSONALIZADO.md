## 📚 Chunks Específicos por Tema

### Tema 1: Séries (Interesse Pessoal)
```
Formal → Real English
"I have watched" → "I've watched" / "I watched"
"What do you think about this?" → "What d'you think about this?" / "Whatcha think?"
"I cannot understand" → "I can't understand" / "I dunno"
"This is very interesting" → "This is so interesting" / "This is sick!"
"I like this character" → "I like this character" / "I'm into this character"
```

**Pronúncia (IPA):**
- watched: /wɑːtʃt/
- character: /ˈkærəktə(r)/
- interesting: /ˈɪntrəstɪŋ/
- episode: /ˈepɪsoʊd/

**Connected Speech:**
- "What did you watch?" → /wɒt dɪd ju wɑːtʃ/ (Linking)
- "I've watched" → /aɪv wɑːtʃt/ (Linking)

---

### Tema 2: Viagens (Objetivo Principal)
```
Formal → Real English
"I would like to book a room" → "I'd like to book a room" / "I wanna book a room"
"Do you have availability?" → "Do you have availability?" / "You got rooms?"
"What is the price?" → "What's the price?" / "How much is it?"
"I need assistance" → "I need help" / "Can you help me out?"
"Where is the bathroom?" → "Where's the bathroom?" / "Where's the restroom?" (US)
```

**Pronúncia (IPA):**
- book: /bʊk/
- room: /ruːm/
- price: /praɪs/
- bathroom: /ˈbɑːθruːm/

**Connected Speech:**
- "I'd like" → /aɪd laɪk/ (Linking)
- "Do you" → /du ju/ ou /dʒu/ (Assimilation)
- "What's the" → /wɒts ðə/ (Elision)

---

### Tema 3: Formação de Frases (Dificuldade Principal)
```
Estrutura: [Sujeito] + [Verbo] + [Objeto] + [Explicação]

Exemplo 1:
"I like this series because the characters are interesting and the plot is complex."

Exemplo 2:
"I want to travel to Europe because I love history and I want to visit museums."

Exemplo 3:
"I think that communication is important because it helps us understand each other."

Padrões Úteis:
- "I think that..." (opinião)
- "For example..." (exemplo)
- "Because..." (razão)
- "In addition..." (adição)
- "However..." (contraste)
```

## 🎤 Exercícios de Pronúncia e Connected Speech

### Exercício 1: Linking (Semana 1)
**Regra:** Quando uma palavra termina em consoante e a próxima começa com vogal, conecte as duas.

```
Praticar:
- "I want to" → /aɪ wɑːn tə/ (não /aɪ wɑːnt tuː/)
- "What is it?" → /wɒt ɪz ɪt/ (não /wɒt ɪz ɪt/)
- "Please ask her" → /pliːz ɑːsk ər/ (não /pliːz ɑːsk hər/)
```

### Exercício 2: Elision (Semana 2)
**Regra:** Quando duas consoantes se encontram, a primeira pode desaparecer.

```
Praticar:
- "next day" → /neks deɪ/ (não /nekst deɪ/)
- "best time" → /bes taɪm/ (não /best taɪm/)
- "last night" → /lɑːs naɪt/ (não /lɑːst naɪt/)
```

### Exercício 3: Assimilation (Semana 3)
**Regra:** Quando dois sons se encontram, o primeiro pode mudar para ficar mais próximo do segundo.

```
Praticar:
- "that girl" → /ðæt ɡɜːl/ muda para /ðæɡ ɡɜːl/
- "don't you" → /doʊnt ju/ muda para /doʊntʃ ju/
- "have to" → /hæv tu/ muda para /hæf tu/
```
