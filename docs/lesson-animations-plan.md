# Plano de Animações por Lição

Este documento lista as animações planejadas para cada lição do Vacation Plus 2, seguindo o padrão estabelecido com "Emily's Texas Adventure".

## Estrutura Base

Cada animação deve ter:
- **6 cenas** com diálogos curtos
- **Duração**: 1-2 minutos
- **Legendas**: Inglês + Português
- **Estilo**: Disney/Pixar 3D
