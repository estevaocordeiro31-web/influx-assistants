## 🎯 O Que Você Vai Aprender

### Em Cada Lição:

1. **Costumes Culturais** - Como cada nacionalidade se comporta em diferentes situações
2. **Vocabulário Regional** - Diferenças entre inglês americano, britânico e australiano
3. **Chunks e Equivalência** - Expressões naturais usadas por nativos
4. **Diálogos Situacionais** - Conversas reais com cada personagem
5. **Exercícios Práticos** - Atividades para praticar o que aprendeu

### Diferenças que Você Vai Dominar:

| Categoria | 🇺🇸 Americano | 🇬🇧 Britânico | 🇦🇺 Australiano |
|-----------|--------------|---------------|-----------------|
| Formalidade | Casual/Direto | Formal/Educado | Super Casual |
| Gírias | "Awesome", "Cool" | "Lovely", "Brilliant" | "No worries", "Mate" |
| Vocabulário | "Sidewalk", "Fries" | "Pavement", "Chips" | "Footpath", "Chips" |
| Expressões | "You got it!" | "Certainly!" | "No drama!" |

## 🚀 Como Usar Este Material

### Para Alunos:

1. **Leia** o material expandido antes ou depois da aula
2. **Pratique** os diálogos em voz alta
3. **Complete** os exercícios de cada lição
4. **Compare** as diferenças entre os três estilos
5. **Escolha** o estilo que mais combina com você!

### Para Professores:

1. Use os diálogos como **material de listening**
2. Crie **role-plays** com os três personagens
3. Discuta as **diferenças culturais** em sala
4. Use as tabelas como **referência rápida**
5. Adapte os exercícios para o **nível da turma**
