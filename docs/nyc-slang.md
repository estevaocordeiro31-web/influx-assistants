# NYC Slang - Expressões Usadas por Jovens em Nova York

## Gírias Essenciais de NYC

| Slang | Significado | Exemplo de Uso |
|-------|-------------|----------------|
| Mad | Muito, bastante | "That's mad cool!" |
| Bet | Concordar, combinado | "Wanna grab pizza? Bet!" |
| Dead ass | Sério, de verdade | "Dead ass, that was the best show" |
| No cap | Sem mentira, verdade | "No cap, this pizza is amazing" |
| Brick | Muito frio | "It's brick outside today" |
| Lit | Animado, divertido | "This party is lit!" |
| Shook | Chocado, surpreso | "I was shook when I saw the price" |
| Buggin | Agindo de forma dramática | "You're buggin if you think that's true" |
| Wildin' | Agindo de forma louca | "He's wildin' at the party" |
| Tight | Legal ou irritado | "That's tight!" / "I'm tight right now" |
| Good looks | Valeu, obrigado | "Good looks for the heads up" |
| Word | Concordância ou surpresa | "Word? That's crazy!" |
| Cop | Comprar algo | "I'm gonna cop those sneakers" |
| Link | Encontrar alguém | "Let's link up later" |
| Bodega | Loja de conveniência | "I'll grab snacks at the bodega" |
| Pie | Uma pizza inteira | "Let's order a pie" |
| Schlep | Carregar algo longe | "I had to schlep all the way downtown" |
| The City | Manhattan | "I'm heading to The City" |
| Straphanger | Passageiro do metrô | "I'm a straphanger every day" |
| Gotchu | Eu te ajudo | "Need a ride? Gotchu!" |
