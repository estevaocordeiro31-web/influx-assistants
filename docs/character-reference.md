## Video Production Process
1. Generate static images for each scene using character references
2. Generate TTS audio with correct accents
3. Apply Ken Burns effects (zoom in/out, pan, shake)
4. Add motion graphics with animated text and emojis
5. Create dual subtitles (English white + Portuguese yellow)

## Image Specifications
- Resolution: 1920x1080 (landscape) for video scenes
- Style: Consistent Disney/Pixar 3D animation
- Character must match reference images exactly
